# 🚀 Cadastro Moderno | UX Experience - Fluxo Code

Este é um projeto de formulário de cadastro de alta fidelidade, focado em **Experiência do Usuário (UX)** e design **Dark Mode**. Desenvolvido com tecnologias puras (HTML, CSS e JS), ele apresenta uma interface elegante, validações em tempo real e total responsividade.

---

## ✨ Funcionalidades

- **Design Glassmorphism:** Estética moderna com transparências e paleta de cores Dark/Blue.
- **Validação Real-time:** Feedback visual imediato para campos obrigatórios, e-mail e senha.
- **Máscara de Telefone:** Formatação automática (Padrão Brasil) enquanto o usuário digita.
- **Toggle Password:** Sistema para mostrar/ocultar a senha com ícone dinâmico.
- **Simulação de Envio:** Botão com estado de "loading" e mensagem de sucesso estilizada.
- **Layout Responsivo:** Adaptado para dispositivos móveis e desktops.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica.
- **CSS3:** Variáveis nativas (`:root`), Flexbox e animações de transição.
- **JavaScript (ES6+):** Manipulação de DOM, Regex para máscaras e API de validação.
- **Font Awesome:** Ícones vetoriais para interface.

---

## 📸 Demonstração

O formulário conta com estados de erro personalizados:

- **Borda vermelha** e mensagem de erro ao perder o foco de um campo inválido.
- **Botão dinâmico** que altera o texto para "Enviando..." após a submissão.
